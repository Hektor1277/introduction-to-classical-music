# 内容许可与版权说明

本文件适用于仓库中的非代码内容，包括但不限于默认手册、截图、示例专栏、默认公开配置和发布文档。

## 1. 适用范围

以下内容属于本文件说明范围，而不是 [LICENSE](LICENSE) 中的代码许可证范围：

- `materials/default-library/**`
- `data/site/config.json`
- `docs/release/**`
- 仓库根部和 `docs/` 中以说明、手册、流程为目的的文档内容

## 2. 默认公开内容

除另有标注外，仓库自带的默认手册截图、示例专栏和公开文档采用以下授权方式：

- 可复制、分发和修改，用于本项目的再发布、文档维护和二次开发说明
- 修改后请保留来源说明，并明确标注已修改

## 3. 不在公开授权范围内的内容

以下内容不因本仓库公开而获得再分发授权：

- 开发者个人资料库、私人条目、私人整理结果
- 外部网站截图、第三方封面图、唱片图片、人物照片
- 通过自动检索或人工收集得到的第三方资源链接所指向内容
- 原始音视频文件、流媒体内容、下载物或其镜像

这些内容的版权和使用条件归各自权利人所有。使用者需自行确认来源合法性与再分发权限。

## 4. 公共发布边界

公共 GitHub 仓库与公共 Release 只应包含：

- 源代码
- 安装版应用程序
- 默认空资料库与使用手册专栏

不得公开包含开发者个人资料库的便携版或其他未完成版权审计的内容包。
