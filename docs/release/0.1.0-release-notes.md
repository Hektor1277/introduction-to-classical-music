# 0.1.0 Release Notes

发布日期：`2026-04-19`

## 首发内容

- 建立公共 GitHub 仓库结构
- 支持 Windows 安装版打包与安装
- 提供空默认资料库与《不全书使用手册》专栏
- 提供桌面启动器、维护工具和版本检索工具集成

## 发布边界

- 公共 Release 只发布安装版
- 开发者个人资料库和私有便携版不公开
- 首发版本不提供应用内自动更新

## 升级与兼容

- 支持同 `AppId` 覆盖升级
- 默认保留 `%APPDATA%\Introduction to Classical Music` 下的设置与日志；安装目录内自带 `library` 会随程序一起安装、升级和卸载
- 公开支持平台为 `Windows 10/11 x64`
