# 0.1.0 Package Manifest

## 公共发布资产

- 源代码仓库
- `不全书 Setup 0.1.0.exe`

## 安装行为

- 安装器类型：`NSIS`
- 安装模式：当前用户级安装
- 程序目录：`%LOCALAPPDATA%\Programs\buquanshu`
- 用户数据目录：`%APPDATA%\Introduction to Classical Music`
- 默认资料库目录：`%LOCALAPPDATA%\Programs\buquanshu\library`
- 升级策略：覆盖安装，保留 `%APPDATA%` 下运行时设置与日志；程序目录默认库会随安装一起更新
- 卸载策略：删除程序文件和程序目录默认库，默认保留 `%APPDATA%` 下运行时设置与日志

## 安装包内关键内容

- 桌面启动器 `不全书.exe`
- 维护工具运行时
- 默认空资料库模板与《不全书使用手册》专栏
- `recording-retrieval-service.exe`

## 不进入公共发布的资产

- 开发者个人资料库
- 个人资料库便携版
- 历史归档、缓存、日志、下载物
