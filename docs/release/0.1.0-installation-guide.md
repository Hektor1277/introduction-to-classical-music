# 0.1.0 安装与启动说明
适用版本：`v0.1.0`

## 公共发布内容

本次公共发布包含：

- 源代码仓库
- Windows 安装版：`不全书 Setup 0.1.0.exe`

本次不公开发布：

- 基于开发者个人资料库构建的便携版

## 源码安装

### 前置环境

- `Windows 10/11 x64`
- `Node.js 22.x`
- `Python 3.13.x`

### 步骤

```powershell
git clone <your-github-url>
cd introduction-to-classical-music-public
npm run bootstrap:windows
npm run doctor:windows
npm run check
npm run build
npm run package:windows
```

## 安装版安装

### 首次安装

1. 运行 `不全书 Setup 0.1.0.exe`
2. 完成当前用户级安装
3. 从开始菜单或 `%LOCALAPPDATA%\Programs\buquanshu\不全书.exe` 启动 `不全书`

### 程序目录与用户数据目录

- 程序目录：`%LOCALAPPDATA%\Programs\buquanshu`
- 用户数据目录：`%APPDATA%\Introduction to Classical Music`
- 默认资料库目录：`%LOCALAPPDATA%\Programs\buquanshu\library`

安装版从 `0.1.0` 开始采用“程序目录内置默认库 + 用户目录保存设置与日志”的结构。程序升级或卸载会处理程序目录，运行时设置与日志默认保留在用户数据目录。

### 启动器入口

- 打开本地构建库
- 打开维护工具
- 打开版本检索工具

## 默认资料库

- 安装版自带一个位于程序目录内的空默认资料库
- 默认库只附带《不全书使用手册》专栏和配套截图
- 不包含开发者个人条目库

## 升级

- 新版本安装器可直接覆盖旧版本
- 升级默认保留 `%APPDATA%\Introduction to Classical Music` 下的设置、日志等运行时数据
- 默认资料库随程序目录一起覆盖更新；如果你在安装目录默认库中维护了内容，请先导出到独立目录再升级
- 如果机器上存在旧版安装目录 `%LOCALAPPDATA%\Programs\introduction-to-classical-music`，安装器会尝试清理旧程序目录并切换到新的 `buquanshu` 程序目录
- 首发版本不提供应用内自动更新

## 卸载

- 安装版提供卸载程序
- 卸载默认删除 `%LOCALAPPDATA%\Programs\buquanshu` 下的程序文件和其中自带的 `library`
- 卸载默认保留 `%APPDATA%\Introduction to Classical Music` 下的设置、日志等运行时数据
- 如果需要保留资料内容，请先在卸载前用维护工具导出资料库

如需彻底清理用户数据，请手动删除 `%APPDATA%\Introduction to Classical Music`。
